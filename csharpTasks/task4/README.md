# Задание по .Net №4
В этом задании вам предстоит реализовать методы контектста данных с использованием Linq-запросов. 

Обычно контекст данных служит объектом для взаимодействия с БД, по сути он представляет собой интерфейс для работы со строками таблицы как с коллекцией.
Здесь же в качестве таблицы выступает просто список объектов - отзывов фильмов (Review). 

Задача проста - реализовать методы-заглушки в ReviewContext.cs и написать нормальные тесты (то что в Main тоже должно проходить)
. Для тестов рекомендуется использовать xUnit.
